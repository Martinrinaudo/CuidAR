# Contribuir a CuidAR

## Convención de commits

Usar el formato: `tipo: descripción corta`

Tipos válidos:
- `feat`: nueva funcionalidad
- `fix`: corrección de bug
- `chore`: mantenimiento, dependencias, limpieza
- `docs`: documentación
- `security`: cambios relacionados a seguridad

Ejemplos:
- `feat: agregar formulario de empleada doméstica`
- `fix: corregir validación de teléfono en solicitud de traslado`
- `chore: limpiar archivos de build del repo`
- `security: agregar rate limiting al login de admin`
